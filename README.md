# I-grec - Réseau Social Nouvelle Génération

Un réseau social moderne et innovant construit avec des technologies web contemporaines.

## 🚀 Fonctionnalités

- 👤 **Authentification par SMS** - Connexion sécurisée
- 💬 **Messagerie en temps réel** - Communiquez avec vos contacts
- 🎥 **Video Call** - Appels vidéo intégrés
- 🎨 **Stories** - Partagez vos moments
- 🔴 **Live Stream** - Diffusez en direct
- 🏆 **Système de badges** - Récompenses et progression
- 🤖 **Assistant IA** - Aide intelligente intégrée
- 📊 **Analytics** - Suivez vos statistiques
- 🌐 **Multilingue** - Support de plusieurs langues
